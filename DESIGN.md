# 设计规范 · Design Guidelines

经期日历 App 的设计规范。当前收录 **间距系统**;后续可继续补充颜色、字号、圆角等。

---

## 03 · 间距系统 (Spacing System)

基于 **4px 基准单位**的间距刻度,确保所有元素的内外边距保持一致的视觉节奏。
**11 级刻度**覆盖从 4px 到 80px 的全部场景。所有间距都以 CSS 变量(令牌)的形式定义在
`index.html` 顶部的 `:root` 中,组件只引用令牌,**不再写死像素值**。

### 刻度与令牌

| 令牌 Token  | 数值  | 典型用途                          |
| ----------- | ----- | --------------------------------- |
| `--sp-1`    | 4px   | 图标与文字、紧邻元素的贴合间隙    |
| `--sp-2`    | 8px   | 标签内边距、chip 间距             |
| `--sp-3`    | 12px  | 卡片内边距、列表项间距            |
| `--sp-4`    | 16px  | **页面外边距(左右留白)**        |
| `--sp-5`    | 20px  | 卡片外边距、区块底部留白          |
| `--sp-6`    | 24px  | 分区标题上间距                    |
| `--sp-8`    | 32px  | 大区块之间                        |
| `--sp-10`   | 40px  | 页面段落之间                      |
| `--sp-12`   | 48px  | 空状态、居中提示                  |
| `--sp-16`   | 64px  | 页面级大间隔                      |
| `--sp-20`   | 80px  | 顶 / 底安全区、超大留白           |

> 注:刻度在小尺寸密、大尺寸疏(4→8→12→16→20→24 之后跳到 32→40→48→64→80),
> 符合视觉节奏,避免相邻档位难以区分。

### 页面外边距统一规则

四个页面共用同一个水平外边距令牌 `--page-gutter`(= `--sp-4` = **16px**):

| 页面           | 统一前         | 统一后            |
| -------------- | -------------- | ----------------- |
| 日历 calendar  | 18px / 30px    | `--page-gutter`   |
| 统计 stats     | 18px           | `--page-gutter`   |
| 热力图 heatmap | 18px           | `--page-gutter`   |
| 我的 me        | 18px           | `--page-gutter`   |

改动来源:此前各页面的外边距各写各的(日历月历网格甚至用 30px),正是
「几个页面间距看起来不一样」的根源。统一到一个令牌后,只要改 `--page-gutter`
一处,四页同时生效。

### 分区与卡片

- **分区标题上间距**:`--sp-6`(24px),下间距 `--sp-2`(8px)—— 见 `.stats-section`
- **卡片内边距**:`--sp-3`(12px)—— 见 `.me-overview`、`.me-btn` 等
- **卡片外边距**:`--sp-4`(16px)

### 使用方式

```css
/* ✅ 引用令牌 */
.some-card { padding: var(--sp-3); margin-bottom: var(--sp-4); }
.some-page { padding-left: var(--page-gutter); padding-right: var(--page-gutter); }

/* ❌ 不要写死像素 */
.some-card { padding: 13px; margin-bottom: 15px; }
```

新增间距时,**先从刻度里挑最接近的一档**,而不是随手写一个新数值。若刻度里
确实没有合适的档位,再讨论是否要扩展刻度本身。

### 有意保留的例外

日历格子(`.day`)、症状圆点偏移等**已调好的微间距**为像素级手工微调,
不纳入 4px 刻度,保持原值不动,避免视觉回归。
